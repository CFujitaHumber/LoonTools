# LoonTools

Set of tools for the Loon-E project, including diagnostics and utilities.

## Description

### Getting Started

Loon Tools is installed by [Loon-Env](https://github.com/HumberASV/Loon-Env) the instructions for which can be found in its [README](https://github.com/HumberASV/Loon-Env/blob/main/README.md). Once Loon-Env is installed, you can run the LoonTools script from anywhere in the terminal.
