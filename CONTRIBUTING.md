# Contributing

You need to create a branch under a naming convention.
Follow the table below for details:

| Prefix | Purpose | Example |
| -------- | -------- | -------- |
| doc | any documentation fixes or features | doc-new-pull-template |
| feat | Any new features or improvements | feat-improved deamon |
| fix | any fixes to bugs | fix-issue-1 |

Create a pull request detailing what you did and why
